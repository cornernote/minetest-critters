Critters
for Minetest

Heaps of critters that run around in the game.

Download GitHub Minetest CoRNeRNoTe

Features

  • 4 groups with dozens of critters.

Docs

Requirements

  • Minetest >=0.4.3

Installation

Download and uncompress the latest version, and move the the minetest-critters-master folder into your minetest/mods folder.
